# Aero Data Engineering Project

## Overview
This project implements a Bronze-Silver-Gold data pipeline using Databricks and PySpark.

## Architecture
- Bronze Layer: Raw data ingestion
- Silver Layer: Data cleaning and transformation
- Gold Layer: Business-level aggregations

## Technologies Used
- PySpark
- Databricks
- Delta Lake
- GitHub

## Pipeline Flow
1. Ingest raw data into Bronze
2. Clean and transform in Silver
3. Build fact and dimension tables in Gold
